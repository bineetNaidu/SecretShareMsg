# SecretShareMsg

Share Messages with Your friends family and other.

Build with Typescript And Base64 Encoding

## Installation

Use Git clone to Clone the app.

```bash
git clone https://github.com/bineetNaidu/SecretShareMsg
cd SecretShareMsg
```
